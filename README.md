# LEXER SQL Compiler
SQL Syntax Checker and Compiler with Lexical Analysis

# Currently Implemented 
- Data Type
    - String
    - Integer
- Statement
    - CREATE DATABASE
    - DROP DATABASE
    - SHOW DATABASES
    - USE DATABASE
    - CREATE TABLE
    - DROP TABLE
    - SHOW TABLES
    - INSERT
    - DELETE
    - UPDATE
    - SELECT

**Currently only supports simple where syntax**
